# Security Policy

## Supported Versions

Only the latest GitHub Release is supported for security fixes.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting Security Issues

Please report security issues through GitHub Issues unless the report contains sensitive live traffic, credentials, private device data, or exploit details that should not be public.

For sensitive reports, contact the maintainer privately before sharing details.

## Scope

SecMP is intended for authorized Android security testing. Use it only on devices, applications, and networks where you have explicit permission.

Captured traffic may contain secrets, tokens, credentials, and personal data. Store captures securely and do not include sensitive captures in public reports.

SecMP stores runtime state and captured traffic locally. It does not upload captured traffic.
